MFFCKRmassanger

Закрытые комнаты для ваших созвонов

Создайте уникальный код, поделитесь им с друзьями и общайтесь текстом и голосом. Приложение готово к бесплатным тарифам Render / Fly / Replit.

Комнаты живут, пока работает сервер. Если он «уснул» на бесплатном хостинге, просто создайте новый код.

Голосовые соединения работают по WebRTC напрямую между браузерами, сервер нужен только для сигналинга.
